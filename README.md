# Metagenomic-Pari-Island
Metagenomic analysis and phylogenetic study of marine bacterial communities from Pari Island, Indonesia.

## Project Overview

This project aims to demonstrate a metagenomic analysis workflow and phylogenetic study of marine bacterial communities from Pari Island, Indonesia.

## Dataset

The dataset is based on the published shotgun metagenomic study of seawater bacterial communities from Pari Island, Indonesia.

BioProject:

* PRJNA1218935
* PRJNA1218910

## Objectives

1. Retrieve microbial sequence data.
2. Perform quality control (QC).
3. Conduct sequence alignment using Clustal Omega.
4. Construct a phylogenetic tree using MEGA.
5. Interpret phylogenetic relationships among dominant marine bacterial taxa.

## Workflow

Data Collection
↓
Quality Control (fastp)
↓
Assembly (metaSPAdes)
↓
Marker Gene Selection (16S rRNA)
↓
Multiple Sequence Alignment (Clustal Omega)
↓
Phylogenetic Analysis (MEGA)
↓
Interpretation

## Software Used

* fastp
* metaSPAdes
* Clustal Omega
* MEGA

## Dominant Taxa Analyzed

* Prochlorococcus marinus
* Synechococcus sp.
* Methylobacterium oryzae
* Candidatus Pelagibacter sp.
* Cognatishimia activa
* Thalassovita sp.
